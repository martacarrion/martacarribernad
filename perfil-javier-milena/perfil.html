Calculadora de Presupuesto

Para la web del salón desarrollé una calculadora interactiva que permite a los clientes estimar el precio de distintos servicios antes de acudir al establecimiento.

La herramienta fue desarrollada utilizando JavaScript, HTML y CSS, gestionando el código mediante GitHub e integrándola directamente en la web del salón.

Diseño de la Web

La página web del salón fue diseñada y desarrollada íntegramente por mí, con el objetivo de crear una presencia digital clara, elegante y adaptada a la identidad visual del negocio.

El diseño prioriza una navegación sencilla, presentación visual del trabajo realizado y herramientas útiles para los clientes, como la calculadora de presupuesto y la integración de contenido audiovisual para redes sociales.
